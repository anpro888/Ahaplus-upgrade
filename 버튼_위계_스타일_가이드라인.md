# AhaPlus v2 — 버튼 위계 및 스타일 가이드라인

## 버튼 위계 분류

```
Level 1  Primary (주요 행동)     — 화면당 1~2개, 가장 중요한 CTA
Level 2  Secondary (보조 행동)   — 취소/닫기 등 Primary의 대안
Level 3  Danger (위험 행동)      — 삭제/환불 등 되돌리기 어려운 행동
Level 4  Outline (일반 행동)     — 수정/보기/상세 등 보조 기능
Level 5  Ghost / Text (최소 강조)— 더보기/접기/링크 등 부가 기능
Level 6  Icon (아이콘 전용)      — 닫기(✕)/뒤로(‹) 등 유틸리티
```

---

## Level 1 — Primary (주요 행동)

**용도**: 저장, 확인, 발송, 등록 등 화면의 핵심 CTA

| 상태 | 배경 | 텍스트 | 테두리 | 기타 |
|------|------|--------|--------|------|
| 기본 | `#6161FF` | `#FFFFFF` | 없음 | |
| **hover** | `#4F4FE0` | `#FFFFFF` | 없음 | `transition: 0.15s` |
| **active/pressed** | `#3D3DC0` | `#FFFFFF` | 없음 | |
| **disabled** | `#6161FF` | `#FFFFFF` | 없음 | `opacity: 0.5; pointer-events: none` |

**hover 패턴**: 배경색 한 단계 어둡게 (Darken)

| 속성 | 값 |
|------|----|
| 높이 | 36~40px (모달 풋터) / 자유 (인라인) |
| padding | `0 24px` (모달) / `10~14px 0` (전체폭) |
| border-radius | `6~8px` (모달) / `10px` (카드형) |
| font-size | 13~14px |
| font-weight | 600~700 |

**적용 예시**: 모달 `저장`/`확인`, 판매 `판매 완료`, 문자 `문자 발송하기`

```css
.btn-primary {
  background: #6161FF; color: #fff; border: none;
  border-radius: 6px; height: 36px; padding: 0 24px;
  font-size: 13px; font-weight: 600;
  cursor: pointer; transition: 0.15s;
}
.btn-primary:hover { background: #4F4FE0; }
.btn-primary:active { background: #3D3DC0; }
.btn-primary:disabled { opacity: 0.5; pointer-events: none; }
```

## Level 2 — Secondary (보조 행동)

**용도**: 취소, 닫기, 이전 등 Primary의 대안 행동

| 상태 | 배경 | 텍스트 | 테두리 |
|------|------|--------|--------|
| 기본 | `#FFFFFF` | `#616161` | `1px solid #BDBDBD` |
| **hover** | `#F5F5F5` | `#616161` | `1px solid #9E9E9E` |
| **active** | `#EEEEEE` | `#424242` | `1px solid #9E9E9E` |
| **disabled** | `#FFFFFF` | `#616161` | `1px solid #BDBDBD` + `opacity: 0.5` |

**hover 패턴**: 배경 연한 회색 + 테두리 한 단계 진하게 (Subtle Fill)

**적용 예시**: 모달 `취소`, 테스트 발송 `취소`

```css
.btn-secondary {
  background: #fff; color: #616161; border: 1px solid #BDBDBD;
  border-radius: 6px; height: 36px; padding: 0 24px;
  font-size: 13px; font-weight: 600;
  cursor: pointer; transition: 0.15s;
}
.btn-secondary:hover { background: #F5F5F5; border-color: #9E9E9E; }
.btn-secondary:active { background: #EEEEEE; color: #424242; }
```

## Level 3 — Danger (위험 행동)

**용도**: 삭제, 환불 등 되돌리기 어려운 파괴적 행동

### 3a. Danger Solid (모달 풋터)

| 상태 | 배경 | 텍스트 | 테두리 |
|------|------|--------|--------|

| **hover** | `#E24B4A` | `#FFFFFF` | 없음 |
| **active** | `#D43A3A` | `#FFFFFF` | 없음 |

**hover 패턴**: 배경색 한 단계 어둡게 (Darken) — Primary와 동일 패턴

**적용 예시**: 예약금 `삭제`, 예약 취소 `확인`

```css
.btn-danger {
  background: #F06060; color: #fff; border: none;
  border-radius: 6px; height: 36px; padding: 0 24px;
  font-size: 13px; font-weight: 600;
  cursor: pointer; transition: 0.15s;
}
.btn-danger:hover { background: #E24B4A; }
.btn-danger:active { background: #D43A3A; }
```

### 3b. Danger Outline (인라인)

| 상태 | 배경 | 텍스트 | 테두리 |
|------|------|--------|--------|
| 기본 | `#FFFFFF` | `#F06060` | `1px solid #F0B0B0` |
| **hover** | `#F06060` | `#FFFFFF` | `1px solid #F06060` |
| **active** | `#E24B4A` | `#FFFFFF` | `1px solid #E24B4A` |

**hover 패턴**: 배경/텍스트/테두리 완전 반전 (Full Invert) — Level 4 Outline과 동일 패턴

**적용 예시**: 예약 상세 `예약 취소`, `환불`

```css
.bd-action-btn.danger {
  color: #F06060; border-color: #F0B0B0;
  background: #fff;

.bd-action-btn.danger:active {
  background: #E24B4A; border-color: #E24B4A;
}
```

## Level 4 — Outline (일반 행동)

**용도**: 수정, 보기, 상세정보, 결제 등록, 재예약 등 보조 기능

| 상태 | 배경 | 텍스트 | 테두리 |
|------|------|--------|--------|
| 기본 | `#FFFFFF` | `#6161FF` | `1px solid #B0B0FF` |
| **hover** | `#6161FF` | `#FFFFFF` | `1px solid #6161FF` |
| **active** | `#4F4FE0` | `#FFFFFF` | `1px solid #4F4FE0` |
| **disabled** | `#FFFFFF` | `#6161FF` | `1px solid #B0B0FF` + `opacity: 0.5` |

**hover 패턴**: 배경/텍스트/테두리 완전 반전 (Full Invert)
- 흰 배경 + 인디고 텍스트 → 인디고 배경 + 흰 텍스트
- 프로토타입 전체에서 가장 많이 사용되는 패턴

| 속성 | 값 |
|------|----|
| padding | `4px 14px` (인라인) / `4px 12px` (테이블) |
| border-radius | `5~6px` |
| font-size | 11~12px |
| font-weight | 500 |

**적용 예시**: 예약 상세 `예약 수정`/`상세정보`/`수정`/`결제 등록`/`예약금 등록`, 예약 내역 `보기`, 고객 카드 `상세 정보`/`재예약`/`예약`, 예약금 결제전 목록 `결제`

```css
.btn-outline {
  padding: 4px 14px; background: #fff;
  border: 1px solid #B0B0FF; border-radius: 5px;
  font-size: 12px; color: #6161FF; font-weight: 500;
  cursor: pointer; transition: 0.15s;
}
.btn-outline:hover {
  background: #6161FF; color: #fff; border-color: #6161FF;
}
.btn-outline:active {
  background: #4F4FE0; border-color: #4F4FE0;
}
```

## Level 5 — Ghost / Text (최소 강조)

### 5a. Ghost (텍스트 버튼, 호버 시 색상 변화)

| 상태 | 배경 | 텍스트 | 테두리 |
|------|------|--------|--------|
| 기본 | 투명 | `#9E9E9E` | 없음 |
| **hover** | 투명 | `#6161FF` | 없음 |
| **active** | 투명 | `#4F4FE0` | 없음 |

**hover 패턴**: 텍스트 색상만 변경 (Color Shift)

**적용 예시**: `+ 자원 추가`

```css
.btn-ghost {
  background: none; border: none;
  color: #9E9E9E; font-size: 12px;
  cursor: pointer; transition: 0.15s;
}
.btn-ghost:hover { color: #6161FF; }
.btn-ghost:active { color: #4F4FE0; }
```

### 5b. Text Link (링크형)

| 상태 | 텍스트 | 장식 |
|------|--------|------|
| 기본 | `#6161FF` 또는 `#D97706` (경고) | 없음 |
| **hover** | 동일 | `text-decoration: underline` 또는 배경 `#EEF0FF` / `#FFE0B2` |

**hover 패턴**: 밑줄 추가 또는 연한 배경 (Underline / Tint)

**적용 예시**: `스팸 필터링 안내 보기`(경고), 고객명 클릭, 더보기

```css
/* 인디고 링크 */
.text-link { color: #6161FF; cursor: pointer; }
.text-link:hover { text-decoration: underline; }

/* 경고 링크 */
.text-link-warn { color: #D97706; cursor: pointer; font-weight: 600; }
.text-link-warn:hover { background: #FFE0B2; border-radius: 4px; }
```

## Hover 패턴 요약 (5가지)

| 패턴 | 설명 | 적용 Level |
|------|------|-----------|
| **Darken** | 배경색만 한 단계 어둡게 | L1 Primary, L3 Danger Solid, 특수(임시저장, 토글) |
| **Subtle Fill** | 투명/흰 배경 → 연한 회색 배경 | L2 Secondary, L6b 뒤로가기 |
| **Full Invert** | 배경↔텍스트↔테두리 완전 반전 | L3 Danger Outline, L4 Outline |
| **Color Shift** | 텍스트 색상만 변경 (회색→인디고) | L5a Ghost |
| **Reveal Fill** | 투명 배경에 배경 나타남 | L6d 유틸리티 아이콘 |

## 모달 풋터 버튼 배치 규칙

```
[취소(L2)]  [삭제(L3, 선택적)]  [저장/확인(L1)]
 Secondary    Danger Solid       Primary
```

- 좌측: 취소 (Secondary, outline)
- 중간: 삭제 (Danger Solid) — 삭제 가능한 경우에만 표시
- 우측: 저장/확인 (Primary, solid)
- 읽기 전용 모달: 단일 `닫기` 버튼 (Secondary)
